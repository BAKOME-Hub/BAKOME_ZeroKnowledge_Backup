❓ FAQ

Is my data really safe?

Yes. Data is encrypted with AES-256-GCM before it leaves your device. The server sees only random noise. Even if the server is hacked, your data remains secure.

What happens if I lose my password?

Your encryption key is derived from your password using Argon2id. Without the password, the data cannot be decrypted. There is no backdoor. Save your password securely.

Which cloud providers are supported?

WebDAV (Nextcloud, ownCloud), Amazon S3 (and MinIO), IPFS (decentralized), and local disk storage.

How is this different from Google Drive?

Google Drive can see your files. With ZK Backup, the server sees only encrypted noise. Zero-knowledge means zero trust required.

Is it free?

Yes! BAKOME ZK Backup is open source under MIT license. Free forever.
