🔐 Client-Side Encryption

Data encrypted BEFORE upload. Server receives only random noise. AES-256-GCM authenticated encryption.

🔑 Zero-Knowledge

Encryption keys derived with Argon2id. Keys NEVER touch the server. Complete privacy guaranteed.

☁️ Multi-Backend

WebDAV, Amazon S3, IPFS, or local disk. Switch between backends anytime.

⚡ Quick Start

$ git clone https://github.com/BAKOME-Hub/BAKOME_ZeroKnowledge_Backup.git
$ cd BAKOME_ZeroKnowledge_Backup
$ cargo build --release
$ ./bakome_zk_backup init webdav://your-server.com/backup
$ ./bakome_zk_backup backup ~/Documents
