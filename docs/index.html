🔐 Encryption

  • AES-256-GCM authenticated encryption
  • Unique key per file
  • Encryption BEFORE upload
  • Server sees only random noise

🔑 Key Management

  • Argon2id key derivation
  • Memory-hard (GPU resistant)
  • 32-byte random salt per file
  • Keys NEVER touch the server

☁️ Storage

  • WebDAV (Nextcloud, ownCloud)
  • Amazon S3 / MinIO
  • IPFS decentralized storage
  • Local disk / external drive

✅ Integrity

  • SHA3-256 hashing per chunk
  • Cryptographic snapshots
  • Tamper-proof verification
  • Complete audit trail

⚡ Quick Start

$ git clone https://github.com/BAKOME-Hub/BAKOME_ZeroKnowledge_Backup.git
$ cd BAKOME_ZeroKnowledge_Backup
$ cargo build --release

$ ./bakome_zk_backup init webdav://nextcloud.example.com/backup
$ ./bakome_zk_backup backup ~/Documents
$ ./bakome_zk_backup restore ./restored
